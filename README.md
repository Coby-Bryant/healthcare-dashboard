# Healthcare Dashboard

A healthcare-focused patient management dashboard built with HTML, CSS, and JavaScript.

## Features

* Patient CRUD operations
* Search and filtering
* Patient status tracking
* Patient notes
* Vital signs management
* Medication management
* Analytics dashboard
* Animated statistics
* Status distribution chart
* Dark and light mode
* Local storage persistence

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## Future Improvements

* Authentication
* Backend database integration
* Real charting library
* Patient history tracking
* Export reports
